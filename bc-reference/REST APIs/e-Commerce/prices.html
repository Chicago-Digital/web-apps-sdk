Prices

    Get prices

    Gets the list of price resources available.

    Request

    • Method: GET
    • Version: v3, this API call supports additional sort and filter mechanisms as well as retrieving only a sub-set of fields

      Note: In order to access the v3 API endpoints please make sure you have enabled the new rendering engine.

    • Server: https://[app key here]-[site_ID here]-apps.worldsecuresystems.com. Take a look at the Authorize your API calls document for more info on how this URL is formed.
      • Note: When building apps always use relative request URLs. Do not use the "full" URL above because you might have problems running your app on a different site as the site_ID parameter will be different.
    • Path: /webresources/api/v3/sites/current/prices
      • Alternatively, use siteID instead of 'current'
    • Authorization header: This should contain the authorization token. Here is how to obtain the token.

    Response

    The details for the specified customer ID in JSON format.

    • {% raw %}productId{% endraw %} - the id of the price (numeric)
    • {% raw %}siteId{% endraw %} - the id of the site (numeric)
    • {% raw %}countryCode{% endraw %} (string)
    • {% raw %}taxCodeId{% endraw %} - the ID of the tax code applied (numeric)
    • {% raw %}wTaxCodeId{% endraw %} - the ID of the tax code applied for wholesalers (numeric)
    • {% raw %}retailPrice{% endraw %} - the retail price (numeric)
    • {% raw %}salePrice{% endraw %} - the sale price (numeric)
    • {% raw %}salePrice2{% endraw %} (numeric)
    • {% raw %}salePrice3{% endraw %} (numeric)
    • {% raw %}quantity2{% endraw %} - the 2-nd quantity threshold (if the volume discounts is enabled) (numeric)
    • {% raw %}quantity3{% endraw %} - the 3-rd quantity threshold (if the volume discounts is enabled)(numeric)
    • {% raw %}wSalePrice2{% endraw %} - the wholesaler sale price for the 2-nd threshold(numeric)
    • {% raw %}wSalePrice3{% endraw %} - the wholesaler sale price for the 3-rd threshold(numeric)
    • {% raw %}wQuantity2{% endraw %} - the 2-nd quantity threshold for wholesalers (if the volume discounts is enabled) (numeric)(numeric)
    • {% raw %}wQuantity3{% endraw %} - the 3-nd quantity threshold for wholesalers (if the volume discounts is enabled) (numeric)(numeric)

    The response is paginated, you can return only a few of the customer's fields. Take a look at the API V3: sorting and filtering for additional switches you can use.

    You can output additional fields to those listed above as default, see the API v3 Discovery APP for additional details.

    Create a new price for a product

    Create a new price for a particular product.

    • Version: v3

    • {% raw %}wSalePrice3{% endraw %} - (double)
    • {% raw %}wQuantity2{% endraw %} - (integer)
    • {% raw %}wQuantity3{% endraw %} - (integer)
